■ 開き方
index.html をダブルクリックすると閲覧できます。
公開時は、フォルダー内のファイル構成を保ったままサーバーへアップロードしてください。

■ フォームについて
このサイトは静的サイトです。加入フォームと問い合わせフォームは、入力内容をブラウザー内で整形し、メールアプリを開く方式です。サーバーへ自動保存されません。
加入フォームには下書き自動保存、入力検証、コピー、TXT保存機能があります。

■ 公開前に必ず確認する項目
1. 「葛西記念財団」という名称と法人格の表示
2. 代表・副代表・本拠点・メールアドレス
3. 定款の内容と実際の運営方法
4. YouTubeチャンネルURL
5. 独自ドメイン設定後のOG画像URL、必要に応じたサイトマップ
6. 史料・画像・文章の権利確認

「財団」は活動団体名として使用し、公益財団法人・一般財団法人等の法人格は表示していません。法人登記、寄附募集、税務、契約等へ正式利用する前に、実態に合わせて専門家へ確認してください。
