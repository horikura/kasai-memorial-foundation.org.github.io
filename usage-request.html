APPLICATION

掲載・利用申請

当サイトの文章・画像・名称等の利用、取材、共同企画について事前にご相談ください。

APPLICATION GUIDE

申請が必要な例

  • 当サイトの文章や画像を冊子、ウェブサイト、動画、展示等へ転載する場合
  • 葛西記念財団の名称、紋章、制作物を企画・広報で使用する場合
  • 取材、撮影、講演、共同企画、出展協力を希望する場合
  • 資料提供や調査協力について正式な回答・確認を希望する場合

引用について

著作権法上認められる適切な引用まで一律に禁止するものではありません。引用の必要性、主従関係、出典表示等を満たす形でご利用ください。判断が難しい場合は事前にご相談ください。

PROCEDURE

申請の流れ

  1. 01

    内容を整理

    利用目的、媒体、使用箇所、公開期間、営利・非営利、希望時期を整理します。

  2. 02

    メールで申請

    必要事項を記載し、参考資料がある場合は添付してお送りください。

  3. 03

    内容を確認

    権利関係、事実関係、活動方針、対応可能性を確認します。

  4. 04

    回答・条件確認

    許可の可否、出典表記、公開範囲等をメールでお知らせします。
